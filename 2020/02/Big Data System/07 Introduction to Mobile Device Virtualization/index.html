07 Introduction to Mobile Device Virtualization

动机：为什么

大规模的物联网设备及其数据需要有效利用云资源
解决方案：虚拟化
但是…隐私问题和性能提升
因此：使物联网设备的虚拟化势在必行

移动设备的虚拟化

虚拟化：在多个操作系统之间，高效的共享物理资源

混合关键设置：通用操作系统（实时OS，安全OS，遗留OS）

减少软件移植费用和硬件成本

主要的组件：硬件设备、Hypervisor（虚拟机监控程序 VMM）、GuestOS（虚拟机 VM）
